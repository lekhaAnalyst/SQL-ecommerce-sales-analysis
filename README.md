# 📊 Global E-Commerce Sales Analysis (2025) using MySQL

## 📌 Project Overview

This project analyzes a 2025 Global E-Commerce Sales dataset using MySQL. The objective is to explore sales performance, identify revenue trends, and generate business insights through SQL queries.

---

## 🛠️ Tools Used

- MySQL Workbench
- SQL
- GitHub

---

## 📂 Dataset Information

- Dataset: Global E-Commerce Sales (2025)
- Records: 1200
- Columns: 7

### Dataset Columns
- Order_ID
- Country
- Category
- Unit_Price
- Quantity
- Order_Date
- Total_Amount

---

## 📚 SQL Concepts Covered

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions (SUM, AVG, COUNT, MIN, MAX)
- CASE Statement
- Date Functions
- Window Function (RANK)

---

## 📈 Business Questions Solved

- Count total orders
- Calculate total revenue
- Find average order value
- Identify highest and lowest value orders
- Analyze country-wise revenue
- Analyze category-wise sales
- Find monthly sales trends
- Identify top-performing countries
- Analyze quantity sold by category
- Rank countries based on revenue

---

## 💡 Key Insights

- Revenue was analyzed across different countries and product categories.
- Top-performing countries were identified based on total sales.
- Monthly sales trends helped understand sales performance over time.
- Product categories were compared using revenue and quantity sold.
- SQL window functions and aggregate functions were used to generate business insights.
